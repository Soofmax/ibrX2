# Accessibilité

Bonnes pratiques utilisées dans le projet:

- Skip link en haut de page
- Focus visible (utilitaires Tailwind)
- Support `prefers-reduced-motion`
- Aria-live pour les mises à jour de la carte
- Contrastes vérifiés sur les éléments principaux

Des audits et améliorations supplémentaires seront documentés ici.
