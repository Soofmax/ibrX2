# Installation

## Prérequis
- Node 20 (voir `.nvmrc`)
- npm 9+

## Étapes
1) Installer dépendances
   npm install

2) Lancer en développement
   npm run dev

3) Construire et prévisualiser
   npm run build
   npm run preview

## Notes
- Mettez à jour Browserslist si demandé:
  npx update-browserslist-db@latest --yes
