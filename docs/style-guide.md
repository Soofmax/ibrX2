# Guide de style

- Stack UI: Tailwind CSS 3.x
- Palette: ambre (CTA), verts foncés (header/footer), neutres stone
- Typo: système (sans serif), polices personnalisées optionnelles
- Animations: modérées, respect de `prefers-reduced-motion`
- Composants: réutilisables, responsives, attention aux doublons (refactor carrousels)

Ajouts à venir: tokens, composants UI génériques, conventions de nommage.
